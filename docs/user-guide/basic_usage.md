# Basic Usage
<hr/>

> Upon opening the app, you are greeted with a map and a text box above. 

<div class="w-full">
    <div class="w-1/4 m-auto">
    ![](/img/user-guide_basic-usage/001.png)
    </div>
</div>

> To start using the app, press the textfield to open the search menu. You can also press the search icon to open/close the menu. 
> If you want to go to the settings, press the gear icon.

<div class="w-full">
    <div class="w-1/4 m-auto">
    ![](/img/user-guide_basic-usage/002.png)
    </div>
</div>

> If you want to use your current geographic location as your place of origin, press the "Press to use your location" button.
> 
> But if you want to manually enter a custom place of origin, start typing a name of a location in the textfield. The textfield currently
> struggles to find places more known by their acronym so it's better if you type the whole name as much as possible.
> 
> Also, if you don't want to press the back icon, you can either use your phone's back function or press the empty space below to exit the search interface.

<div class="w-full">
    <div class="w-1/4 m-auto">
    ![](/img/user-guide_basic-usage/003.png)
    </div>
</div>

> After selecting a location as your origin location, another textfield will show up. That is for inputting your desired destination.

<div class="w-full">
    <div class="w-1/4 m-auto">
    ![](/img/user-guide_basic-usage/004.png)
    </div>
</div>

> For selecting the destination location, just type the name of your destination the same way as the origin.
:::info
Due to Nominatim API's nature, for example, typing "AUF" will not work. You have to type "Angeles University" for proper results to show up.
:::

<div class="w-full">
    <div class="w-1/4 m-auto">
    ![](/img/user-guide_basic-usage/005.png)
    </div>
</div>

> After selecting your destination, the app will shortly compute for possible shortest routes.
> Wait for a few seconds and the shortest path options will appear.

<div class="w-full">
    <div class="w-1/4 m-auto flex-row">
    ![](/img/user-guide_basic-usage/006.png)
    </div>
</div>

:::info
If you accidentally closed the interface, press the "View Searched Routes" button to return
:::

    
<div class="w-full">
    <div class="w-1/4 m-auto flex-row">
    ![](/img/user-guide_basic-usage/007.png)
    </div>
</div>
